## For users

1. Перейти на банковские репозиторий с пакетами 
```
registry=https://nxrepo.fuib.com/repository/fuib-npm-group/
strict-ssl=false
sass_binary_site=https://nxrepo.fuib.com/repository/node-sass
```
2. Установить пакет
```
npm i @mill/ng
```
